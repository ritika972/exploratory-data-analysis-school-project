- Exploratory Data Analysis (EDA)
- Feature Engineering
- Linear Regression Model

## Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Outcome:
Derived meaningful insights related to student performance and school infrastructure.
